# Catarina API
API de dados pessoais
- Dados pessoas
- Dados endereco
- Dados telefone


## Description

Não precisem mais criar tabela de clientes, empresas, endereço, telefones, etc em seus sitemas. Use o Catarina-API

## Installation

```bash
$ npm install
$ docker run --name mysql -p 3306:3306 -e MYSQL_ROOT_PASSWORD=root -d mysql
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Documentação
http://localhost:3000/api

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
